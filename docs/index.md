## Easy to get started

Manually create the first document

```bash
//Create a directory for component development
$ mkdir library && cd library

// install dumi
$ npm i dumi

// create document
$ mkdir docs && echo '# Hello dumi!' > docs/index.md

// preview the document
$ npx dumi dev
```
